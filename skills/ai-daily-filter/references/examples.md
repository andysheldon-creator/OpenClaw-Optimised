## 可补选情况

当天重要推文只有1-2条时，以下可考虑补选：
- Mistral 客户案例
- NVIDIA 量子加速
- Grok 排名截图（作为八卦）
- 大公司幕后故事（如果内容有价值）
- **实用工具类**：如 ArXiv 论文分类工具等，虽是小功能点，但对 AI 主要用户（程序员、科研人员）实用

## 排序对比示例

### 正式发布 vs 预览

- Claude in Excel（正式发布）→ 优先级更高
- Claude Cowork（预览）→ 优先级较低

### 第一个实现 vs 跟进者

- 其他公司先有10秒视频 → 优先级高
- Grok 视频延长到10秒 → "并不是第一家实现的"，优先级低

### 影响面

- Excel 用户量极大 → 优先级高
- Skills 热点但影响小于 Excel → 优先级较低

### 附带内容

- Hinton 附带视频 → 优先级高于 Bengio
- Bengio 无视频 → "内容没有hinton带视频的厚"

### 观点正向性

- Andrew Ng "对大多数群体有正向建议作用" → 优先级高
- LeCun "并不被所有人接受" → 优先级低
