# Opciones de APIs de IA Gratuitas (2026)

## ✅ Opciones Recomendadas (Verdaderamente Gratuitas)

### 1. Google Gemini (Google AI Studio)

**Estado**: ✅ Tier gratuito sin tarjeta de crédito

**Modelos disponibles**:

- Gemini 2.5 Pro
- Gemini 2.5 Flash
- Gemini Flash-Lite

**Límites del tier gratuito**:

- 5-15 solicitudes por minuto (RPM)
- 100-1,000 solicitudes por día (RPD)
- 250,000 tokens por minuto (TPM)
- Context window: 1 millón de tokens

**Ventajas**:

- No requiere tarjeta de crédito
- Excelente para desarrollo y pruebas
- Modelos potentes y confiables
- Integración con Google AI Studio

**Cómo obtener API key**:

1. Ir a [Google AI Studio](https://aistudio.google.com/)
2. Crear cuenta con Google
3. Generar API key gratuita

---

### 2. Groq (GroqCloud)

**Estado**: ✅ Tier gratuito sin tarjeta de crédito

**Modelos disponibles**:

- Llama 3.1 8B: 14,400 req/día, 6,000 tokens/min
- Llama 3.3 70B: 1,000 req/día, 12,000 tokens/min

**Límites del tier gratuito**:

- Miles de solicitudes diarias
- Créditos promocionales: $25 iniciales + $150/mes (programa de datos)

**Ventajas**:

- Extremadamente rápido
- No requiere tarjeta de crédito
- Modelos open-source (Llama)
- Generoso tier gratuito

**Cómo obtener API key**:

1. Ir a [GroqCloud](https://console.groq.com/)
2. Crear cuenta
3. Generar API key gratuita

Para recursos limitados y uso gratuito, las mejores opciones son:
