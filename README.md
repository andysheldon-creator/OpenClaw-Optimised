# OpenCray

这是一个 OpenClaw 的定制版本，专注于增强国内 IM 渠道的支持。

## 主要特性

- **飞书 (Feishu) 集成**: 深度优化的飞书机器人支持，支持消息接收与回复。
- **国内 IM 适配**: 针对国内常用的即时通讯工具（如飞书、企业微信等）进行功能增强。
- **网络优化**: 针对国内网络环境的连接适配。

## 快速开始

本项目基于 [OpenClaw](https://github.com/openclaw/openclaw) 构建。基础功能请参考官方文档。

### 开发指南

1. 安装依赖：
   ```bash
   pnpm install
   ```

2. 启动开发模式：
   ```bash
   pnpm dev
   ```

3. 构建项目：
   ```bash
   pnpm build
   ```
